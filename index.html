PORTIFÓLIO

Jogo da Velha

Tecnologias

  • HTML
  • CSS
  • JavaScript

Este projeto é uma versão interativa do clássico Jogo da Velha (ou Tic-Tac-Toe), desenvolvido com HTML, CSS e JavaScript puro, sem uso de bibliotecas externas. O foco foi treinar lógica de programação, controle de estado do jogo e manipulação dinâmica da interface com o DOM. A interface permite que dois jogadores se enfrentem alternando entre os símbolos X e O, com verificação automática de vitória, empate e reinício da partida.

Calculadora

Tecnologias

  • HTML
  • CSS
  • JavaScript

Este projeto é uma calculadora funcional desenvolvida com HTML, CSS e JavaScript puro, com o objetivo de reforçar os conceitos fundamentais de manipulação do DOM, eventos e lógica condicional. A calculadora realiza as quatro operações básicas (adição, subtração, multiplicação e divisão) e possui uma interface limpa e responsiva. Durante o desenvolvimento, trabalhei com event listeners, validação de entrada, e controle de estado da operação em andamento.

Projeto Android

Tecnologias

  • HTML
  • CSS

Este projeto consiste na criação de uma página institucional fictícia para um sistema Android, desenvolvida com foco em estruturação semântica, estilização visual e organização de conteúdo em seções. Foi desenvolvido utilizando HTML5 e CSS3 puros, com objetivo de praticar layout responsivo, posicionamento de elementos e hierarquia de títulos. O site simula uma apresentação de produto, contendo cabeçalho, seção de imagem principal, área de informações, galeria de imagens e rodapé.
